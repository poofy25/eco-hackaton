# ReMatero — Product Vision

## Problem Statement

Construction and demolition sites across Romania generate thousands of tons of surplus materials annually — bricks, lumber, steel, tiles, fixtures, hardware — that end up in landfills or illegal dump sites. There is no standardized, trustworthy marketplace for these materials. Today's "market" is fragmented across:

- **WhatsApp/Facebook groups** — no search, no quality guarantees, no traceability
- **Phone calls & word-of-mouth** — limited reach, no price transparency
- **OLX/marketplace generalists** — poor categorization for construction materials, no domain-specific features

This creates a triple loss: **sellers** pay disposal fees for perfectly usable materials, **buyers** overpay for new materials they don't need, and the **environment** absorbs unnecessary waste and emissions.

## Target Audience

### Sellers
| Segment | Description | Volume |
|---------|-------------|--------|
| **Construction companies** | Surplus from overordering or project changes | Medium-High (pallets, bulk) |
| **Renovation contractors** | Leftover materials from residential/commercial jobs | Low-Medium (mixed) |
| **Demolition firms** | Recovered materials from teardowns | High (structural, bulk) |
| **Individuals** | Post-renovation leftover materials | Low (single items) |

### Buyers
| Segment | Description | Motivation |
|---------|-------------|------------|
| **Small construction firms** | Cost-conscious contractors | 30-70% savings on materials |
| **Individual renovators** | DIY homeowners | Budget-friendly renovation |
| **NGOs & schools** | Community projects, repairs | Limited budgets, social impact |
| **Architects & designers** | Upcycling / reclaimed material projects | Unique materials, sustainability |

## Value Proposition

**"Materialele bune nu se aruncă — se reutilizează."** *(Good materials aren't thrown away — they're reused.)*

| vs. Current alternatives | ReMatero advantage |
|--------------------------|-------------------|
| No quality standards | 3-tier certification system (Basic → Verified → Pro) |
| No searchability | Category tree + location + condition filters |
| No trust/accountability | Profiles, ratings, reviews, verified sellers |
| No traceability | Reservation system, transaction history |
| No logistics support | Partner integration for transport (Phase 2) |
| No impact measurement | CO2/waste savings tracked per transaction |

## Competitive Advantage

1. **Domain-specific taxonomy** — material categories designed for construction (not generic "Home & Garden")
2. **Trust & certification** — verification tiers that give buyers confidence in material quality
3. **Circular economy metrics** — every transaction shows environmental impact (kg diverted, CO2 saved)
4. **Geolocation-first** — proximity search because construction materials are heavy and expensive to transport
5. **B2B + B2C** — serves both companies (bulk) and individuals (small quantities)

## MVP Scope (Phase 1 — 4-6 Weeks)

### In Scope
- User registration & authentication (email + phone)
- Seller profiles with basic verification
- Listing CRUD with photos, description, category, condition, price, location
- Search & filter (category, location radius, condition, price range)
- Reservation system (buyer reserves, seller confirms)
- In-app messaging (buyer ↔ seller)
- Basic rating system (post-transaction)
- Responsive web app (mobile-first)

### Out of Scope (Phase 2+)
- Logistics partner integration
- Payment processing
- Advanced certification (Verified/Pro tiers)
- Mobile native apps
- AI-powered categorization
- Analytics dashboard

## Pilot Plan

### Phase: Pilot (Weeks 1-2 post-launch)
- **3-5 partner companies** in Bucharest metro area (pre-recruited)
- **Target:** 50-100 active listings within first 2 weeks
- **Focus:** seller onboarding experience, listing quality, buyer discovery

### Success Criteria for Pilot
| Metric | Target |
|--------|--------|
| Active listings | 50-100 |
| Registered users | 100-200 |
| Reservations made | 15-25 |
| Completed transactions | 5-10 |
| Avg. time to first reservation | < 7 days |
| Seller satisfaction (survey) | > 3.5/5 |

### Pilot Partners Profile
- 1 large construction company (high volume, diverse materials)
- 1 demolition firm (bulk structural materials)
- 1-2 renovation contractors (mixed residential materials)
- 1 individual seller (validate B2C flow)
